# 📰 Hindi & Hinglish Fake News Detector

A dual-embedding based machine learning project designed to detect fake news claims in pure Hindi and code-mixed Hinglish text.

## 🚀 Features
* **Bilingual Support:** Understands and processes both Hindi and Hinglish.
* **Dual-Embedding Architecture:** Combines XLM-RoBERTa and Google's MuRIL for deep contextual understanding.
* **Real-time Web UI:** User-friendly interactive interface built with Flask, HTML, and CSS.

## 🛠️ Tech Stack
* **Backend:** Python, Flask
* **Machine Learning:** PyTorch, HuggingFace Transformers
* **Models:** XLM-RoBERTa (Cross-lingual), MuRIL (Indian Languages)
* **Frontend:** HTML, CSS, JavaScript

## ⚙️ How to Run
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the Flask server: `python app.py`
4. Open your browser and go to `http://127.0.0.1:5000`
